W3C-"Regras" para a web

  • W3C (Word Wide Web Consortium) – um consorcio internacional, ou seja, uma organização sem fins lucrativos que reúne empresas para desenvolver padrões e diretrizes para a web.
    O css foi criado pelo mesmo em 1996, juntamente com o html que foi criado em 1991
  • Na verdade, o W3C não impõe regras, ele cria padrões técnicos.
  • Garantem interoperabilidade — o mesmo código funciona em todos os navegadores e dispositivos.
  • Asseguram acessibilidade — tornando a web utilizável por pessoas com deficiência.
  • Guiam a evolução da web — formalizando tecnologias amplamente usadas e definindo seu futuro.
