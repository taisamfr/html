O QUE É

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar elementos em uma página da web.
Com isso, é possível separar o conteúdo (HTML) da sua representação visual (CSS).
A relação entre HTML e CSS é essencial, pois o HTML é a base de um site e o CSS representa a sua parte estética.
CSS (Cascading Style Sheets) = folhas de estilo em cascata
